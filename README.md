# 👑 魔王軍入団面接アプリ（Streamlit × OpenAI）

このアプリは、魔王が面接官となり、ユーザーの「魔王軍入団適性」を判定する対話型ゲームです。  
Streamlit と OpenAI API を使用して構築されています。

## 📦 機能概要

### 🔥 魔王軍入団面接モード
- 魔王が威圧的な質問を投げかける  
- 最大 10 ターンの面接  
- 3 ターン以降、回答が十分なら魔王が評価  
- 評価基準（各25点）
  - 野心  
  - 忠誠心  
  - 戦闘力  
  - 論理性  
- 最終的に **合格 / 不合格** を判定
